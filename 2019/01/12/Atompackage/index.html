Atom编辑器插件

发表于 2019-01-12 | 次阅读

Atom,一款好用的文本编辑器 我认为他会是未来的趋势！

Github 出品的基于 Electron / Node.js 的半开源文本编辑器

开源！注意他是开源的，这就意味着全世界在使用他的人如果有意愿，都可以升级改进它！
GitHub 上的大佬叫他是 Hackable 就不是吹的，配置文件是 cson，样式文件是 CSS 和 Less，所以理论上你可以把 Atom 配置成任何你想要的样子。

主题：

自带四款主题，随心搭配，黑色白色棕色？自己选！

语言方面：

刚刚安装好Atom，全是英文看不懂怎么办 直接点击install a package搜索Chinese！


因为我已经装好中文插件，所以可以看到我的界面已经是中文了！
想用其他语言的也可以安装别的，点击后面的Disable或者Eable就可以启用或者禁用了

美化方面

界面美化

Activate-power-mode


除了图片以上特效：还有：屏幕右上角的连击数，达到一定连击之后的音效！可以自己在配置文件中更改，选择关闭或启用任何一种特效。

Atom clock

在编辑器右下角显示当前时间，全屏模式看时间必备！

编辑简化：

Color picker

windows:ctrl + alt + c
Mac :command + shift + c

Emmet

使用Emmet语法快速编写HTML。

autocomplete-paths

写路径时自动补全

goto-definition

直接跳转到函数（变量）的定义处，支持Hyperclick即Ctrl/Command + 鼠标左键直接跳转，不过需要根据文档自行设置
