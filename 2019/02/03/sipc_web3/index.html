科技协会web组第三次培训——假期小提升第一周

发表于 2019-02-03 | 更新于 2019-04-05 | 次阅读

相信通过两周的自学大家已经对web前端和php的基本语法有了一定了解，但是我们知道只有一个网页如果没有交互，他是没有生机的，他是没有生命的。所以，请仔细阅读前两次后端培训记录（已经把实验部分补全）：

  • 第一次后端培训。主要内容：初识php和在前端实现一个简单的表单
  • 第二次后端培训。主要内容：将表单与后台数据库连接并美化表单

假期作业

借助前两次培训内容，在你的前端页面上加入用户注册，登录的功能，要求使用pdo的方法操作数据库。
不反对借鉴复制培训给出的代码（反正培训代码也不是用pdo操作的数据库哈哈哈哈）。
其他部分只要知道代码啥意思即可

理智分析

…

…嘶…只需要实现登录注册，那么我们需要必须掌握的知识有：

  • 表单
  • php获得表单数据
  • php连接数据库
  • php操作数据库
  • sql语句 INSERT
  • sql语句 SELECT
