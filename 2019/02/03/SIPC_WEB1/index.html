php语言和我们掌握的C语言的不同？

  • 代码块均有<?php ?>包裹
  • 弱类型语言啥玩儿全用$定义！

学生名字叫小明学号为123456

爽！但并不代表他没有类型！在基本变量中她还比C语言多了两种类型：’字符串’(String) 布尔型（Boolean）

  • 数组，不，应该叫做集合
    php的数组是一个疯狂的东西，里面啥玩儿都能放！

Array ( [name] => 小明 [number] => 123456 )
array(2) { [“name”]=> string(6) “小明” [“number”]=> int(123456) }

注意在数组最后一个值后面加 ‘,’ 只有在最新的php7.3中才开始支持(“number” => 123456,)

  • 新的循环方式foreach

[one] => 1
[two] => 2
[three] => 3

  • 庞大丰富的函数库
  • 支持面向对象
